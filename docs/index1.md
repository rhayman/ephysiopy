# Home

A set of tools for the analysis of electrophysiological data recorded
using [Axona](http://www.axona.com/) or [OpenEphys](https://open-ephys.org/) data, be that using tetrode or [Neuropixels](https://www.neuropixels.org/)

## Installation

ephysiopy requires python 3.13 or greater. The easiest way to install is using uv or pip.

Install [uv](https://docs.astral.sh/uv/)


Then, to install ephysiopy, run:

```bash
uv pip install ephysiopy
```

or with pip:

```bash
python3 -m pip install ephysiopy
```
